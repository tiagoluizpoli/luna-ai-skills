# Design Curator Skill

Semantic Design System Curator. Generates agent-friendly DESIGN.md files that enforce premium, anti-generic UI standards.

## What This Skill Does

- Defines the **visual DNA** for any project
- Generates production-grade `DESIGN.md` files
- Enforces **anti-slop** standards (banned neon, banned system fonts, banned linear easing)
- Provides template for color, typography, motion, and component standards
- Works directly with **Stitch** for AI screen generation

## Resources

Content migrated from `taste-design` with all existing resources preserved. See `resources/` directory for any reference material.

## Quality Gate

Every design system artifact must:
- Use ≤ 2 accent colors
- Derive typography from a mathematical scale
- Use spring physics for motion
- Document explicit anti-patterns
- Include dark mode token mapping
