# Tailwind Architect Skill

Tailwind CSS v4.x architect. Expert in CSS-first configuration, deprecated class migration, and zero-overhead utility systems.

## What This Skill Does

- **Migrates** every deprecated v3 class to its v4 equivalent
- **Architects** design token systems using `@theme` blocks
- **Creates** custom utilities and variants with `@utility` / `@variant`
- **Optimizes** builds leveraging the Rust-based Oxide engine
- **Enforces** zero `!important`, zero `@apply`, zero legacy config
